1.新建一个ReviewClient类 eg: rc = ReviewClient()

2.get_his_p()方法用来获取历史用户列表。eg：rc.get_his_p(_number,_person_num, _time_begin, _time_end, _name)，   1）其中第一个参数为每次获取用户的数量（长度为2的字符串）；
  2）第二个参数为用户ID（长度为18的字符串，如果不提供id，则传入‘000000000000000000’）；
  3）第三个参数为起始时间（格式为‘YYYY-MM-DD hh:mm:ss', 如果不提供该时间，则传入‘0000-00-00 00:00:00’；
  4）第四个参数为结束时间（格式为‘YYYY-MM-DD hh:mm:ss', 如果不提供该时间，则传入‘0000-00-00 00:00:00’；
  5）第五个参数为名字（字符串，如果不提供则传入‘None’）
 
  eg: 查询2012年10月1日到2日的用户，每次返回10个
      rc.get_his_p('10','000000000000000000','2012-10-01 00:00:00','2012-10-02 00:00:00','None')
      查询名字为赵鹏飞的孕妇所有记录，每次返回10个
      rc.get_his_p('10','000000000000000000','0000-00-00 00:00:00','0000-00-00 00:00:00','赵鹏飞')
      查询身份证号为123456789012345678的用户所有记录，每次返回10个
      dc.get_his_p('10','123456789012345678','0000-00-00 00:00:00','0000-00-00 00:00:00','None')
 
  返回的结果会存入self.his_patient属性中（字典），key是用户所有信息的一个元组，value为uuid。界面显示需要遍历该字典所有key，然后读出来显示。

3. 当选择某个孕妇某次记录后，开始传输该次记录数据，调用方法为rc.review(_uuid),参数为第二步所获得字典中某一项   的value.

4. self.his_data=[] 存储数据（跟实时情况格式相同）
   self.his_note=[] 存储注释（跟实时情况格式相同）
   用户信息为鼠标选取某项后，相对应的his_patient项的key
